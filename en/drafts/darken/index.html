Darken

Darken is a simple JavaScript bookmarklet that dims the page you are browsing for night reading. Unlike other solutions, it doesn't invert colors or otherwise alter page content. Everything stays the same, only less bright.

Try now

Tested in Chrome and Safari on a Mac and iOS. Darken is only able to dim the content part of a browser window, while the address bar and buttons remain bright (however in some cases you can work around this by switching to fullscreen mode). This functionality should really be built into OS, but is’s not for now — the lowest brightness level is still too bright at night. There are apps and standalone browsers that have lower brightness settings, but I prefer to keep it simple.

Installation

Dektop or laptop:

Just drag the following link to your bookmarks bar: Darken.

iOS and other mobile devices:

  1. Select and copy the code below:

    javascript:(function(){if(window.isDimmed===true){document.getElementsByTagName('html')[0].style.cssText='filter:none;-webkit-filter:none;-moz-filter:none;';window.isDimmed=false;}else{document.getElementsByTagName('html')[0].style.cssText='filter:brightness(0.3);-webkit-filter:brightness(0.3);-moz-filter:brightness(0.3);-webkit-transform:translateZ(0);';window.isDimmed=true;}})();

  2. Create a bookmark on this page and save it.

  3. Edit the bookmark and paste the code you’ve previously copied into the bookmark’s address field. Save.

The bookmark will automatically sync to all of your devices where you have syncing enabled.

Usage

Once you feel like the Internet is too bright for your eyes, just click “Darken” and see the magic happen.
